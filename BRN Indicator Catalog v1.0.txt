# 목적

BRN은 뉴스를 모으는 서비스가 아니다.

BRN은 대한민국 부동산 시장을 구성하는 다양한 데이터를
수집·표준화·분석하여
현재 시장과 미래 시장을 예측하는 플랫폼이다.

본 문서는 BRN에서 사용하는 모든 지표를 정의한다.

# BRN 내부 Score

모든 지표는 0~100점으로 변환한다.

100점 = 매우 긍정
80점 = 긍정
60점 = 중립
40점 = 부정
20점 = 매우 부정

# BRN 철학

뉴스는 시장을 설명한다.

시장은 데이터를 설명한다.

투자는 데이터를 이해하는 사람의 것이다.

BRN은
데이터 → 시장 → 뉴스 → AI
순서로 시장을 설명한다.
